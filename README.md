# Pokedex 
Projeto de uma Pokedex, em Single Page Application (SPA), alimentada pela PokéAPI, usando React.JS. Nele, pode-se visualizar uma lista com um card de todos os pokemons, onde clicando em cada card, se é redirecionado a uma página contendo informações adicionais sobre o respectivo pokemon.
## Ferramentas utilizadas
Nesse projeto foi utilizado, Browser router, React Hooks (useState, useEffect), useContext e styled components.
